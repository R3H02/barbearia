# Barbearia

Projeto de site dinâmico para uma barbearia desenvolvido com PHP, HTML e CSS.

## Sobre

Este repositório contém uma aplicação web para uma barbearia chamada *Barbearia Moderna*. O projeto foi criado para praticar o uso de PHP para gerar páginas dinâmicas, processar formulários e organizar conteúdo entre diferentes partes do site.

A aplicação inclui páginas como:

- Página inicial com descrição dos serviços
- Agendamento de horários
- Cadastro de clientes
- Contato  
- Componentes reutilizáveis por meio de `includes` de PHP

As páginas usam PHP para renderizar conteúdo e organizar a navegação.

## Tecnologias utilizadas

O projeto foi desenvolvido com:

- **PHP** para lógica no servidor
- **HTML** para estrutura das páginas
- **CSS** para estilos e layout
- Possível suporte ao servidor local (como XAMPP, LAMP, WAMP, etc.)
